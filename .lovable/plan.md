

## Улучшение админ-панели

Админ-панель уже существует по адресу `/admin` (логин: `admin@admin.com`, пароль: `admin123456`). Сейчас там есть базовое управление заказами и товарами. План — сделать её значительно удобнее.

---

### 1. Улучшение раздела «Заказы» (AdminOrders)

**Что добавим:**
- Фильтрация по статусу (новый / в работе / доставлен / отменён)
- Поиск по имени клиента или телефону
- Раскрывающаяся карточка заказа — клик по строке показывает полную информацию: комментарий, дата/время доставки, детальный состав
- Цветовые индикаторы статусов (зелёный — доставлен, жёлтый — в работе, красный — отменён)
- Кнопка удаления заказа

### 2. Улучшение раздела «Товары» (AdminProducts)

**Что добавим:**
- Фильтрация по категории и наличию
- Поиск по названию
- Drag-and-drop загрузка фото (перетаскивание файла в зону)
- Превью фото в списке товаров крупнее
- Подтверждение перед удалением товара (диалог «Вы уверены?»)
- Возможность быстро переключать «В наличии» прямо из списка без открытия формы

### 3. Дашборд (новая страница)

**Новый раздел `/admin/dashboard`:**
- Общее количество заказов и товаров
- Количество заказов по статусам
- Сумма продаж (всего и за сегодня)
- Быстрые ссылки на заказы и товары

---

### Технические детали

**Файлы для изменения:**
- `src/pages/admin/AdminOrders.tsx` — фильтры, поиск, детали заказа, цветовые статусы
- `src/pages/admin/AdminProducts.tsx` — фильтры, поиск, drag-drop фото, быстрое переключение наличия, подтверждение удаления
- `src/pages/admin/AdminDashboard.tsx` — новый файл с дашбордом
- `src/pages/admin/AdminLayout.tsx` — добавить ссылку на дашборд в навигацию
- `src/App.tsx` — добавить маршрут `/admin/dashboard`

**Без изменений в базе данных** — все данные уже есть в таблицах `orders` и `products`.
