## 🌷 Сайт продажи тюльпанов — Москва

### Концепция дизайна

Лёгкий, весенний, романтичный стиль. Мягкие пастельные тона (розовый, лавандовый, кремовый, мятный), плавные анимации, округлые формы. Уход от стандартной «сетки карточек» — акцент на визуальную историю, крупные фото, мягкую типографику.

---

### Бэкенд (Supabase / Lovable Cloud)

**Таблицы:**

- `products` — название, описание, цена, фото (URL), категория, в наличии
- `orders` — имя, телефон, адрес доставки, комментарий, статус, дата, состав заказа (JSON)
- `user_roles` — роли для доступа к админке (admin)

**Аутентификация** — email/пароль для админа через Supabase Auth

**Storage** — загрузка фото товаров

---

### Страницы сайта

1. **Главная** — полноэкранный герой-баннер с цветами, блок «Почему мы», хиты продаж (карусель), CTA-блок заказа
2. **Каталог** — фильтр по категориям (букеты, штучные, композиции), красивые карточки с hover-эффектом
3. **Карточка товара** — крупное фото, описание, цена, кнопка «В корзину»
4. **Корзина** — список выбранных товаров, изменение кол-ва, итоговая сумма
5. **Оформление заявки** — форма (имя, телефон, адрес, дата/время доставки, комментарий), без онлайн-оплаты
6. **Доставка** — условия, зоны, стоимость доставки по Москве
7. **Политика конфиденциальности** — стандартный текст
8. **Оферта** — публичная оферта

### Компоненты

- **Хедер** — логотип, навигация (Каталог, Доставка), иконка корзины с бейджем количества
- **Футер** — контакты, ссылки на документы (оферта, политика), соцсети, копирайт

### Админка (защищённая авторизацией)

- **Страница логина** — email/пароль
- **Панель заявок** — таблица всех заказов, смена статуса (новый → в работе → доставлен)
- **Управление товарами** — добавление/редактирование/удаление, загрузка фото
